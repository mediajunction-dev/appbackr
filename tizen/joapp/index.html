What is ?

Jo is a thin candy shell for your HTML5 apps.

Jo was designed for mobile first. It's a free, lightweight JavaScript framework. Developers create a native-like app experience with Jo for iOS, Android, webOS, BlackBerry, Chrome OS & now Tizen. Jo is designed to create applications, it uses HTML5 as a development stack and does not require direct DOM manipulation.

What is ?

Tizen will help HTML5 developers monetize their apps.

Tizen is an open-source and standard-based software and based on the Linux operating system. It's backed by hardware makers that are building devices running the Tizen Platform. These devices need apps.

FOR THE USER

Tizen is an open ecosystem and compatible with all browsers. It offers a consistent experience across all devices for the user

FOR THE DEVELOPER

Tizen provides developers a new operating system to source and distribute their apps.
